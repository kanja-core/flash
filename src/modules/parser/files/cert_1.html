bigmac.

An economic indicator?

Purpose
The dataset has prices over time of the popular MacDonalds sandwich. The idea is that because you can get it around the world it serves as an interesting economic indicator.
Source
This dataset was originally found in the economist.
Contents

Name Description
date The date of the measurement.
currency_code Internatinal code for the currency in the country
name Name of the country.
local_price Price of a bigmac in the local currency.
dollar_ex The dollar exchange rate.
dollar_price The price of a bigmac translated back to dollars.
Download

You can download the dataset from the commandline.


wget https://calmcode.io/static/data/bigmac.csv

You can also download it directly.
